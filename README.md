# Dog Cat Transformer

## Description
A simple http endpoint that takes arbitrary JSON payloads, and replaces any references of the value “dog” with “cat” (but not any strings containing dog).

### Tech Stack
- Node.js
- [NestJS](https://nestjs.com)
- TypeScript
- Jest 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
